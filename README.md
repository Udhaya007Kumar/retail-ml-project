# 🛒 Retail Sales ML Project

Machine Learning project to predict retail sales using multiple models.

## 📊 Models
- 📈 **Regression** → Sales amount predict (XGBoost R²=0.9988)
- 🏷️ **Classification** → High/Medium/Low predict (XGBoost 99.4%)
- 🔵 **Clustering** → Store grouping (KMeans)
